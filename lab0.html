The AVR toolchain sanity check lab

This lab was a simple AVR toolchain sanity check, making sure we have every process for AVR development ready for the actual labs for Embedded systems.

Largely, I took the time this lab to create templates where prossible, for example, I couldn't find a makefile either on d2l or elsewhere, so using some snippets and some knowledge I made my own. This should allow me to simply make a local copy of the lab 0 repository from github, change the necessary parameters, then begin coding. Likewise for the class website presented here, to tape and staple this template together from some grifter selling webdev courses I made a post template which would only require a simple writeup and edit to the homepage for basic presentability.

Below you should find my video with all requirements as listed inside the Lab 0 assignment whitepaper, as well as the Github repository for quick pulldown.
